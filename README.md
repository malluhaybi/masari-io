## 🚀 خطوات الرفع

### الطريقة 1: Hostinger (الدومين) + Vercel (الاستضافة)

1. ارفع الملفات على GitHub
2. اربط GitHub مع Vercel
3. في Hostinger: وجّه DNS لـ Vercel
4. أضف OPENAI_API_KEY في Vercel

### الطريقة 2: Hostinger فقط

1. ارفع كل ملفات HTML مباشرة عبر File Manager
2. الـ API لن يعمل (يحتاج Vercel أو backend)
